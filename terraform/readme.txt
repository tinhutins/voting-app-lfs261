aws connecting to eks: https://hub.qovery.com/guides/tutorial/how-to-connect-to-your-eks-cluster-with-kubectl/

download kubectl
download aws cli and run aws configure (provide region aws_key and id)
check identity - aws sts get-caller-identity
download aws-iam-authenticator
add iam user to admins
download kubeconfig file (generated by terraform)
set the context for kubectl : export KUBECONFIG=<path to the kubeconfig file you downloaded>
kubectl get nodes
kubectl get pods -n kube-system


gcloud connecting to gke : 

gcloud init
gcloud auth application-default login
